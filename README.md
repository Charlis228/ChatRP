# ChatRp

ChatRp is a modern roleplay chat mod for Minecraft Fabric, based on the AdvancedChat framework and redesigned for immersive RP gameplay.

## Features

- Player Heads integration in chat
- Nickname search with TAB completion
- Unlimited chat history
- Chat history saved between sessions
- Dedicated RP tab
- Russian localization
- Improved chat management
- Better player mention support
- Modern user interface
- Roleplay-focused functionality

## Installation

1. Install Fabric Loader.
2. Install Fabric API.
3. Place ChatRp into the mods folder.
4. Launch Minecraft.

## Supported Versions

- Minecraft 1.21.11
- Fabric Loader
